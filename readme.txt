Extend Fusion with an Instgram element.

== Description ==

The Instagram Element supports the embedding of Instagram feeds.

**Requires the core [Fusion](https://wordpress.org/plugins/fusion/) plugin and either the [Instagram Feed](https://wordpress.org/plugins/instagram-feed/) plugin or [Instagram Feed Pro](https://smashballoon.com/instagram-feed/) plugin to be installed, activated and configured for the feeds desired.**

== Installation ==

1. Ensure that you have already installed the [Fusion core](https://wordpress.org/plugins/fusion/) plugin.
2. Install Fusion : Extension - Instagram through the WordPress plugins screen, or upload the plugin files to the `/wp-content/plugins` directory.
3. Activate Fusion : Extension - Instagram through the 'Plugins' screen in WordPress
4. Visit the Fusion knowledge base and user guide at <http://agencydominion.helpscoutdocs.com/>

== Frequently Asked Questions ==

= Does the Fusion : Extension - Instagram plugin require the Fusion core plugin to function? =

Yes. The Fusion : Extension - Instagram plugin is an extension for adding a Instagram element to Fusion. The Fusion core plugin must be installed for this plugin to function. You can get Fusion [here](https://wordpress.org/plugins/fusion/).

= Will Fusion work with my theme? =

Yes. Fusion is designed to be compatible with themes that adhere to WordPress' theme development standards.

= How can I get more Fusion features? =

We have designed the Fusion core plugin to be fast, light, and easily expandable. We have also built a number of extension plugins so that you choose which functionality you would like to add based on the needs of your website.
